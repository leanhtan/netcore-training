#  Simple Blog Engine

Simple Blog Engine is a website created base on .NET Core

## Technology

### Font-end
* SPA with Vue.js
* Axios for call api 

### Back-end
* ASP.NET Core MVC
* Razor Pages
* Swagger using Swashbuckle
* Microsoft Account external login
